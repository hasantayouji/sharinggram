# sharinggram

Flutter Instagram Clone

Download apk: https://drive.google.com/open?id=1agl4MV5AeVr3YRVM2WlD0mzLnIwr4KMJ

Features:
- Backend using firestore
- Authentication using Firebase Auth
- Realtime notification and timeline update
